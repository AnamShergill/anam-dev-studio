# ⚡ Quick Start Guide

## 🚀 Get Your Portfolio Running in 5 Minutes

### Step 1: Update Your Info (2 min)
Open `data/portfolio.ts` and update:
```typescript
name: "Your Name"
email: "your.email@example.com"
github: "https://github.com/yourusername"
linkedin: "https://linkedin.com/in/yourusername"
```

### Step 2: Add Images (2 min)
Add to `public/` folder:
- `profile.jpg` - Your photo
- `resume.pdf` - Your resume
- `projects/*.jpg` - 7 project screenshots

### Step 3: Run Dev Server (1 min)
```bash
npm run dev
```
Open http://localhost:3000

## 🎯 Your 7 Projects Are Ready!

All projects are already configured with:

- ✅ Placeholder URLs

## 📝 Update Project URLs

Find each project and replace:
```typescript
liveUrl: "https://your-actual-url.vercel.app"
githubUrl: "https://github.com/yourusername/repo-name"
```
