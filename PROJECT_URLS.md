## 🔄 How to Update URLs

1. Open `data/portfolio.ts`
2. Find each project object
3. Replace `liveUrl` with your actual deployed URL
4. Replace `githubUrl` with your actual GitHub repository URL

Example:
```typescript
{
  id: 1,
  title: "Levy Consultant Tax Firm",
  liveUrl: "https://your-actual-url.com",  // ← Change this
  githubUrl: "https://github.com/your-username/your-repo"  // ← Change this
}
```
